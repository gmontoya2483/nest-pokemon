# Ejecutar en desarrollo

1. Tener Nest CLI instalado
   ```
   npm i -g @nestjs/cli
   ```
2. Clonar el repositorio
3. Instalar las dependencias
   ```
   npm install
   ```
4. Levantar la base de datos
   ```
   docker-compose up -d
   ```

## Stack usado

- Nestjs
- MongoDB
